# PROTESES_DE_GUERRA

PROTESES_DE_GUERRA é um jogo 2D desenvolvido utilizando GameMaker e a linguagem GML (GameMaker Language).  
O projeto foi criado com foco em aprendizado e experimentação no desenvolvimento de jogos, explorando mecânicas de movimentação, combate e gerenciamento de vida do personagem.

---

## 🎮 Sobre o Jogo

O jogo apresenta um sistema de gameplay baseado em ação 2D, onde o jogador controla um personagem capaz de se movimentar pelo cenário, enfrentar inimigos e sobreviver utilizando habilidades e ataques.

Durante o desenvolvimento foram aplicados conceitos importantes de desenvolvimento de jogos, como:

- movimentação de personagem
- sistema de gravidade e pulo
- sistema de combate
- gerenciamento de vida
- lógica de gameplay utilizando eventos
- organização de objetos e recursos

## ⚙️ Mecânicas Implementadas

O projeto atualmente conta com sistemas como:

### 🕹 Sistema de Movimento
- movimentação horizontal
- gravidade
- pulo do personagem

### ⚔ Sistema de Combate
- ataques do jogador
- detecção de colisão
- aplicação de dano

### ❤️ Sistema de Vida
- vida máxima do jogador
- redução de vida ao receber dano
- interface de barra de vida
